This is a extra widget for the Leaflet Map plugin

== Description ==

There is a real good plugin called "Leaflet Map" in the repository. Unfortunately the developers missed to create a simple widget for this.
This plugin is filling the gap.

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/leaflet-map-widget` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to widgets and use it.


== Frequently Asked Questions ==

= Is this running alone ? =

No, you need to install the plugin "Leaflet Map"
